# Action

## Import Action

With a local file  :

```bash
$ cds action add --url $GOPATH/src/github.com/ovh/cds/contrib/actions/actions/cds-git-clone.hcl
```

With a remote file  :

```bash
$ cds action add --url https://raw.githubusercontent.com/ovh/cds/master/contrib/actions/cds-git-clone.hcl
```
