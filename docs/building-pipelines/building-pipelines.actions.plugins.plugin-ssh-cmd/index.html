plugin-ssh-cmd

This plugin helps you to run cmd on remote server over ssh.

Parameters

  • hostnames: Hostnames (comma separated values)
  • command: Command
  • timeout: Timeout (seconds)
  • commandTimeout: Command Timeout (seconds)
  • username: Username
  • privateKey: SSH RSA private key
