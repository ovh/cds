plugin-group-tmpl

This actions helps you generate a marathon group application file. It takes a config template file as a single application, and creates the group with the variables specified for each application in the applications files. Check documentation on text/template for more information https://golang.org/pkg/text/template.

Parameters

  • config: Template file to use
  • output: Output path for generated file (default to .out or just trimming .tpl extension)
  • applications: Applications file variables
