Integrations

An integration enables some features on CDS.

It can concern the storage of the artifacts, the repositories manager, the hooks available to trigger workflows, the infrastructure used to spawn the workers.

Here are the extensions available:


Storage


  • AWS S3
  • OpenStack Swift


Hooks on CDS Workflows


  • Kafka Hooks
  • RabbitMQ


Application Deployment


  • Kubernetes Deployment


Repositories Managers


  • Bitbucket Cloud
  • Bitbucket Server
  • Gerrit Repository Manager
  • GitHub Repository Manager
  • GitLab Repository Manager


Infrastructure used by CDS Workers


  • Docker Swarm
  • Kubernetes Compute
  • Openstack Compute
  • vSphere


CDS Events


  • Kafka CDS Events


User Authentication


  • GitHub Authentication
  • GitLab Authentication
  • LDAP Authentication
  • OpenID-Connect Authentication


Artifact Manager


  • Artifactory
